# Eu Sou Programador

Um aplicativo simples desenvolvido em **Flutter**, criado com o objetivo de estudos e prática da estrutura básica de interfaces mobile. O app exibe um título no topo e uma imagem centralizada na tela.

## Funcionalidades

* Exibição de um título personalizado na `AppBar`.
* Renderização de imagem centralizada utilizando `Assets`.
* Estrutura base limpa e organizada.

## Aprendizados

Este projeto é ideal para quem está iniciando e deseja entender:
* Estrutura base de um app Flutter (`MaterialApp`, `Scaffold`).
* Configuração e uso da `AppBar`.
* Widgets de layout básicos (`Center`, `Column`).
* Gerenciamento de **Assets** e imagens no `pubspec.yaml`.

## Tecnologias utilizadas

* [Flutter](https://flutter.dev/)
* [Dart](https://dart.dev/)

## Como executar o projeto

### Pré-requisitos

Antes de começar, você vai precisar ter instalado em sua máquina:
* [Flutter SDK](https://docs.flutter.dev/get-started/install)
* Um emulador Android/iOS ou um dispositivo físico configurado.

### Rodando a aplicação

```bash
# Clone este repositório
git clone [https://github.com/seu-usuario/eu-sou-programador.git](https://github.com/seu-usuario/eu-sou-programador.git)

# Acesse a pasta do projeto no terminal/cmd
cd eu-sou-programador

# Instale as dependências
flutter pub get

# Execute a aplicação
flutter run
